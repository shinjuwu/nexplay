# frontend Vue 3 + Vite

The project uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

## Recommended IDE Setup

- [VS Code](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
[manager-web] npm run manager-dev
[agent-web  ] npm run agent-dev
```

### Compiles and minifies for production-dev

```
[manager-web] npm run build-manager-dev
[agent-web  ] npm run build-agent-dev
```

### Compiles and minifies for production-qa

```
[manager-web] npm run build-manager-qa
[agent-web  ] npm run build-agent-qa
```

### Compiles and minifies for production-ete-client

```
[manager-web] npm run build-manager-ete-client
[agent-web  ] npm run build-agent-ete-client
```

### Deploying a static site for production files

```
[manager-web] npm run preview-manager
[agent-web  ] npm run preview-agent
```
